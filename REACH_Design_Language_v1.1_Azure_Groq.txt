Key Patterns Extracted
Navigation: Left Vertical Sidebar — Zoho, Salesforce, HubSpot, Keap all moved to a persistent left sidebar in their 2024–2025 redesigns. REACH follows this.
Color: Dark Nav + Light Canvas — HubSpot, Pipedrive, Close — dark sidebar (#1A2332) contrasted with near-white content area (#F8FAFC). Creates visual hierarchy without hard chrome.
Typography: Inter / DM Sans — Pipedrive, Close, Salesforce. Inter is the de-facto CRM typeface for legibility at small sizes and data density.
Components: Cards + Status Badges — All 12 CRMs use elevated white cards for primary content units and pill-shaped status badges for pipeline/delivery states.

5.  Layout & Grid System
REACH uses a fixed left sidebar + fluid content canvas layout — the dominant pattern across HubSpot, Salesforce, Zoho NextGen, and Pipedrive. The grid inside the canvas is 12-column with 24px gutters. Deployed on Azure Static Web Apps with responsive breakpoints at 1024px, 1280px, and 1440px.

10.  Tailwind CSS Implementation
All design tokens map to Tailwind CSS custom theme extensions in tailwind.config.js. This ensures zero custom CSS for standard components and consistent rendering on Azure Static Web Apps.

// tailwind.config.js
module.exports = {
  theme: { extend: {
    colors: {
      'ink':    { navy: '#0F1B2D' },
      'brand':  { blue: '#1A73E8', green: '#00B386', coral: '#FF6B4A' },
      'surface':{ canvas: '#F8FAFC', card: '#FFFFFF', panel: '#F1F5F9' },
      'border': { default: '#E2E8F0' },
      'text':   { primary: '#0F1B2D', secondary: '#64748B', muted: '#94A3B8' },
      'status': { success: '#15803D', warning: '#92400E', danger: '#DC2626' }
    },
    fontFamily: {
      sans: ['Inter', 'system-ui'],
      display: ['DM Sans', 'Inter'],
      mono: ['Courier New', 'monospace']
    }
  }}
}
